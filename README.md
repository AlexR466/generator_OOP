# Generator OOP

Программа генерирует фейковые данные, добавляет их в файл форматов CSV, XLSX 
и создает архив данного файла.
Предусмотрена возможность создания многотомного архива.


## Usage/Examples

```bash
pip install -r requirements.txt
python main.py
```
